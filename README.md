# The Red House with Hexagonal Windows

A short 2D point-and-click mystery game made in Unity 2D.

## Story
Mark leaves home to buy groceries and comes back 
to find his house is completely gone...

## Made with
- Unity 2D
- Hand-drawn original sprites and art
